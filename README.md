# クエストダンジョン - 目標達成RPG (PWA)

私生活・仕事の目標を「ダンジョンマップ」にして、スゴロク感覚でステップ(マイルストーン)をクリアしていくスマホ向けPWAアプリ。

![アイコン](icons/icon-192.png)

## コンセプト

1. **ビジュアル重視** — ドット絵RPG風のマップで達成度がひと目でわかる
2. **操作は最小限** — 目標作成は3項目だけ。名前やメモの編集は自動保存
3. **裏目標** — ゴールの先に「裏ゴール」を設定し、さらなる高みを目指せる

## 主な機能

### ダンジョンマップ
- ゴール(目標)・裏ゴール(裏目標)・ステップ数を入力すると、曲がりくねった道のマップを自動生成
- ステップは**好きな順番で**クリア可能。タップ→「クリアした!」だけ
- クリア時はズームイン+パーティクル演出、勇者がマップを進んでいく
- ゴールには**宝箱**、裏ゴールには**豪華な宝箱**。裏ゴールはゴール達成まで霧に隠されている
- すべてのステップをクリアするとゴールの宝箱が開けられる(紙吹雪+バナー演出)
- 各ステップ・ゴールには画像を埋め込み可能

### 記録帳
- マップ右上の本アイコンから開くジャーナル
- 投稿日は自動付与。文字+画像(複数可)を時系列で格納
- 例: 筋トレの記録として毎日写真をアップロードする

### 実績(ホーム画面)
- ゴール達成で**金の星**、裏ゴール達成で**紫の星**が星棚に並ぶ
- マップごとの進捗バーとステップ数を一覧表示

## 技術構成

- **ビルド不要の静的PWA** — Vanilla JS (ES Modules) + CSS のみ、依存ライブラリなし
- **オフライン完全対応** — Service Worker がアプリシェルとフォントをキャッシュ
- **データはすべて端末内** — IndexedDB(マップ・記録・画像Blob)。サーバー不要
- **ドット絵はコード生成** — `js/sprites.js` の文字グリッド→SVG変換(画像アセット不使用)
- フォント: [DotGothic16](https://fonts.google.com/specimen/DotGothic16)(SIL OFL 1.1、`fonts/` に同梱)

## 動かし方

静的ファイルを配信するだけで動きます。

```bash
# ローカルで試す
python3 -m http.server 8000
# → http://localhost:8000 をスマホ/ブラウザで開く
```

GitHub Pages 等の静的ホスティングにそのまま置けば、スマホから「ホーム画面に追加」でアプリとしてインストールできます。

## ディレクトリ構成

```
index.html            アプリシェル
manifest.webmanifest  PWAマニフェスト
sw.js                 Service Worker(オフラインキャッシュ)
css/style.css         ドット絵RPG風スタイル
js/
  app.js              ルーター + SW登録
  db.js               IndexedDBラッパー
  sprites.js          ドット絵スプライト(グリッド→SVG)
  util.js             共通ユーティリティ
  views/
    home.js           ホーム(星棚+マップ一覧)
    wizard.js         マップ作成ウィザード
    map.js            ダンジョンマップ(演出・宝箱・霧)
    records.js        記録帳
fonts/                DotGothic16(OFL、オフライン用に同梱)
icons/                PWAアイコン(tools/gen-icons.mjs で生成)
tools/gen-icons.mjs   依存なしのPNGアイコン生成スクリプト
```

## アイコンの再生成

```bash
node tools/gen-icons.mjs
```
